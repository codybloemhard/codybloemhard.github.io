Ringwm

Summary: A bespoke window manager with a novel UX written in Rust with Penrose.

Stack: Rust

Year: 2024, 2025


Ringwm in based on rings and super swallowing. The screen is split up in to at most two columns; in contrast with other window managers. Each column is a ring of windows. New windows are placed in a ring and swallow the previous window on top. The goal is to be so efficient you only need a few columns and to give windows room to work with.
