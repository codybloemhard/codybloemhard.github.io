Shapebar

A status bar that can display a selection of separators.

Stack: C

Year: 2020


Shapebar is based on lemonboy/bar and krypt-n/bar which provides the standard Lemonbar functionality along with Xft support. It can draw seperators such as triangles, sigmoid, hemisphere. You can set the width and height for the seperators to stretch them to a size you like. It also now has a clone mode where it copies the bar over your screens.

Procedural Terrain Generation

Real scale generated terrain with different biomes.

Stack: C#, Unity3D

Year: 2016


It aimed to present the player with kilometers of landscape on a realistic scale. So the mountains were kilometers high, as they are in real life. It has a dynamic LOD that is mulithreaded. It has a grass biome, a snow biome and a desert biome. There are also generated 3D models placed around the terrain, following the player. This includes trees, bushes, different kind of stones and cacti.
