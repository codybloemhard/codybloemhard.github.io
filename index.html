Utrecht Companion to the Earth

An app that helps geoscience students learn in the field.

Stack: Android Studio, Kotlin, Rust

Year: 2020


Utrecht Companion to the earth is an app that has the goal to aid Geo-science students. Our client for the projectwas the Geo-science department of Utrecht University. My responsibility was optimizing large amounts of data for mobile use and building the Android app. Technologies used by me were Rust and Kotlin with Android Studio. I learned about building Android apps, Shapefile and KML files, compression and the Geo-science field.

Shapebar

A status bar that can display a selection of separators.

Stack: C

Year: 2020


Shapebar is based on lemonboy/bar and krypt-n/bar which provides the standard Lemonbar functionality along with Xft support. It can draw seperators such as triangles, sigmoid, hemisphere. You can set the width and height for the seperators to stretch them to a size you like. It also now has a clone mode where it copies the bar over your screens.

Raytracer

A cpu raytracer in C#, made with other people.

Stack: C#, OpenGL

Year: 2018


Made by Egor Dmitriev, Erwin Glazenburg and me. It features spheres, planes, triangles, .obj models, point lights, spotlights, stochastic area lights(soft shadows), stratisfied sampled lights, textures, HDR skyboxes, reflection, refraction, stochastic glossy reflections, anti-aliasing(SSAA and FXAA), multithreaded rendering, responsive rendering.

Linux Rice

My Linux configuration installed with a script.

Stack: Linux, Shell, C

Year: 2019


This is a script you can and it will install and configure your system to be just like mine. There is an maintained Artix/Arch Linux version and an old Void Linux version. The major components are i3, dmenu, shapebar, st. Programs such as St, Dmenu, shapebar have patches applied and code edited. It supports live theme switching.

Generate Midi Music

Learns from a midi file and than generates music.

Stack: C++

Year: 2017


It reads a midi file of existing music and 'learns' from it. It uses a markov chain and processes information like what notes follow each other, what chords come with notes etc.

Procedural Terrain Generation

Real scale generated terrain with different biomes.

Stack: C#, Unity3D

Year: 2016


It aimed to present the player with kilometers of landscape on a realistic scale. So the mountains were kilometers high, as they are in real life. It has a dynamic LOD that is mulithreaded. It has a grass biome, a snow biome and a desert biome. There are also generated 3D models placed around the terrain, following the player. This includes trees, bushes, different kind of stones and cacti.
