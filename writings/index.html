Incodoc

This website is made using incodoc-ssg, a creation of mine. Incodoc stands for incorporeal document. The corporeal version is presented in simple HTML+CSS and contains no Javascript. No cookies, no trackers, no ads, no analytics. For more info on incodoc please see:
